The ``MyList`` module
==========================

Using ``MyList``
---------------------

First import ``MyList`` from the ``MyList`` module:

	>>> MyList = __import__('1-my_list').MyList

Now use it:

	>>> my_list = MyList([1, 2, 4, 3, 5])
	>>> print(my_list)
	[1, 2, 4, 3, 5]
	>>> my_list.append(6)
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5, 6]
	>>> print(my_list)
	[1, 2, 4, 3, 5, 6]
